What is DocDigitizer Sync?

DocDigitizer Sync transforms your PDF documents into structured, actionable data. Simply upload a document and receive extracted information in JSON format.

Key Features

  • OCR Extraction - Extract text from scanned documents using Google Cloud Vision
  • Document Classification - Automatically identify document types (invoice, receipt, contract, etc.)
  • AI-Powered Extraction - Extract structured fields using advanced language models
  • Multi-Document Processing - Automatically split and process PDFs with multiple documents

Quick Start

1

Get Your API Key

Request an API key at docdigitizer.com/contact

2

Send Your First Document

curl -X POST https://apix.docdigitizer.com/sync \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "file=@invoice.pdf"
3

Receive Extracted Data

Get structured JSON with classified document type and extracted fields.

Tip: Include the X-DD-TraceId from the response when contacting support for faster resolution.

Understanding Results

Response Structure

{
  "requestToken": "ABC1234",
  "state": "Completed",
  "pipeline": "MainPipelineWithOCR",
  "pageCount": 3,
  "output": {
    "extractions": [
      {
        "documentType": "Invoice",
        "fields": {
          "invoiceNumber": "INV-2024-001",
          "date": "2024-01-15",
          "totalAmount": 1250.00,
          "vendorName": "Acme Corp"
        }
      }
    ]
  },
  "timers": {
    "total": 2345.67,
    "ocr": 1200.00
  }
}

Response Fields

FieldDescription
requestTokenUnique trace ID for this request
stateProcessing status: Completed, Failed, Partial
pipelinePipeline used for processing
pageCountNumber of pages processed
output.extractionsArray of extracted documents
timersProcessing time breakdown (ms)

Multi-Document Results

When a PDF contains multiple documents, each is returned as a separate entry in the extractions array with its own document type and fields.

PowerShell Tools

For Windows users, we provide a PowerShell module for easy integration.

Installation

# Set your API key
$env:DOCDIGITIZER_APIKEY = "your-api-key-here"

# Import the module
Import-Module .\cmdlets\DocDigitizer.psd1

Available Commands

CommandDescription
Send-DocDigitizerDocumentProcess a PDF document
Test-DocDigitizerConnectionTest API connectivity
Get-DocDigitizerConfigView current settings
Set-DocDigitizerConfigUpdate settings
Get-DocDigitizerHelpShow help

Usage Examples

# Process a single document
Send-DocDigitizerDocument -FilePath "invoice.pdf"

# Process and save result to JSON
Send-DocDigitizerDocument -FilePath "invoice.pdf" -SaveExtraction

# Process all PDFs in a folder
Get-ChildItem *.pdf | Send-DocDigitizerDocument -SaveExtraction

Troubleshooting

Common Issues

Document not recognized

  • Ensure the document is clear and readable
  • Check that text is not too small or blurry
  • Verify the PDF is not corrupted

Timeout on large documents

  • Large documents take longer to process
  • Consider splitting very large PDFs
  • Contact support for high-volume processing

Missing fields in extraction

  • Some fields may not be present in all documents
  • Field names may vary by document type
  • Contact support for custom extraction schemas

Authentication errors

  • Verify your API key is correct
  • Ensure the key is included in the X-API-Key header
  • Check if your key has expired
